# 🍷 Wine Prediction AI Service

Advanced Machine Learning web application using the Scikit-learn `load_wine()` dataset with an interactive Gradio interface.

---

## 🚀 Features

- AI-based wine classification
- Feature selection using 3 features:
  - Alcohol
  - Malic Acid
  - Ash
- Interactive slider-based UI
- Confidence score prediction
- AI probability analysis visualization
- Modern dashboard layout using Gradio
- Public web deployment support

---

## 🛠 Technologies Used

- Python
- Scikit-learn
- Gradio
- Pandas

## 📊 Dataset

This project uses the `load_wine()` dataset from Scikit-learn for wine classification prediction.

### Selected Features
- Alcohol
- Malic Acid
- Ash

The machine learning model was trained using `RandomForestClassifier` with feature selection preprocessing.
